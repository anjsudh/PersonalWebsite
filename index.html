Download this project as a .zip file Download this project as a tar.gz file

A Personal Website that provides Windows 8 like UI. Uses BootMetro for designing UI. Provides options for presenting personal details, a contacts page, interests and even one's resume in a very colourful and appealing manner. Uses gmaps.js for maps and mCustomScrollbar for customizing the scrollbar for all div tags. Demo available at http://www.anjanasudhir.co.nf
